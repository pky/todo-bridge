**レイアウト構成**:

**コンポーネント**:

#### サイドバー
- スマートリストセクション（折りたたみ可能）
- リストセクション（折りたたみ可能）

- リスト追加ボタン

#### タスク一覧
- タスク追加入力欄（Smart Add対応）
- タスク項目（チェックボックス、優先度、名前、タグ、期限）
- 完了タスクは取り消し線 + グレーアウト

### 6.5 操作の違い

| 操作 | デスクトップ | モバイル |
|------|-------------|---------|
| タスク完了 | チェックボックスクリック | チェックボックスタップ or スワイプ右 |
| タスク削除 | 詳細パネルで削除 | スワイプ左 or 詳細画面で削除 |
| タスク詳細 | 右パネルに表示 | フルスクリーン遷移 |
| リスト切替 | サイドバークリック | ハンバーガーメニュー→タップ |
| 検索 | ヘッダーの検索バー | 🔍アイコンタップ→検索バー展開 |

### 6.6 タッチ操作（モバイル）

```
タスク項目のジェスチャー:
← スワイプ左: 削除（赤背景表示）
→ スワイプ右: 完了（緑背景表示）
タップ: 詳細画面へ遷移
長押し: 複数選択モード（将来対応）
```
